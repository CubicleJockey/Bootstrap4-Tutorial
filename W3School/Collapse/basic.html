Collapse

The .collapse class indicates a collapsible element (a <div> in this example); this is the content that will be shown or hidden with a click of a button.

To control (show/hide) the collapsible content, add the data-toggle="collapse" attribute to an <a> or <button> element. Then add the data-targert="#id" attribute to connect the button with the collapsible content (<div id="demo">).

Note: for <a> element use the href attribute instead of the data-target attribute. Note: by default collapsible content is hidden.
