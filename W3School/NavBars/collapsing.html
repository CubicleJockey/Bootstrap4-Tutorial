Basic Navigation Bars

To create a collapsible navigation bar, use a button with .navbar-toggler class. data-toggle="collapse" and data-target="#theTarget". Then wrap the navbar content (links, etc) inside a <div> element with a class of .collapse navbar-collapse, followed by an id that matches the data-target of the button: "theTarget":
