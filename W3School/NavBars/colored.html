Basic Navigation Bars

Use any of the .bg-color classes to change the background color of the navbar ( .bg-primary, .bg-secondary, .bg-success, .bg-warning, .bg-danger, .bg-info, .bg-dark and .bg-light)

Tip: Add the White text color to all links in the navbar with the .navbar-dark class, or use the .navbar-light class to add black text color.
